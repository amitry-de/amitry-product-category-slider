# Amitry Product Category Slider

A lightweight WooCommerce plugin to create responsive product and category sliders.

## 🚀 Features

- WooCommerce Product Slider
- WooCommerce Category Slider
- Gutenberg Support
- Elementor Support
- Shortcode Support
- Lightweight & Fast
- SEO Friendly
- Mobile Responsive
- Swiper loaded locally (GDPR friendly)

## Installation

Install directly from WordPress.org or upload the plugin manually.

👉 https://wordpress.org/plugins/amitry-product-category-slider/
